# PriorityClass

## Contexte

Vous travaillez dans un cluster avec un deployment existant `busybox-logger`. Le cluster a déjà au moins une PriorityClass user-defined.

## Objectifs

- Comprendre les PriorityClasses
- Créer une nouvelle PriorityClass avec la bonne valeur
- Appliquer une PriorityClass à un deployment existant
